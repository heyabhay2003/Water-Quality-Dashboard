#  IoT Based Water Quality Management System

This dashboard is a part of my NABARD project on **IoT Based Water Quality Management System**. It is designed to monitor important water quality parameters in real time and display them through a simple, user-friendly dashboard.

## Features

- Live monitoring of water quality
- pH, Temperature, EC and TDS values
- Water level monitoring
- Device status and battery information
- Alert section for abnormal readings

## Technologies Used

- HTML
- CSS
- JavaScript
- ESP32

## Purpose

The main aim of this project is to help farmers and researchers monitor water quality easily using IoT technology. The system is intended to provide real-time information and support better water management.
